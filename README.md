# Лабораторная работа №9. Создание собственного приложения на основе Jetpack Compose
## Цель работы
Создать полноценное Compose-приложение (фильмы / аниме /
игры — на ваш выбор), используя знания всех предыдущих лабораторных работ:
компонуемые элементы, состояние, обработку событий, работу с макетами,
переключателями, кнопками, изображениями и поддержку ориентации экрана.
## Логические секции
* область изображения
* блок с описанием
* панель управления (кнопки переключения)
## Используемые элементы
* Column / Row
* Image
* Text
* Surface
* выравнивание и отступы
* Modifier для размеров
